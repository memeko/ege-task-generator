# Генератор заданий 19-21 ЕГЭ

Страница генерирует одну игровую задачу с тремя связанными вопросами 19, 20 и 21.

Что есть на странице:
- блок теории по позициям `В1`, `П1`, `В2`, `П2`;
- единое условие игры;
- три подпункта 19, 20 и 21;
- карта начальных позиций;
- спойлер с пошаговым разбором и кодом на Python.

## Запуск

Откройте файл `index.html` в браузере или поднимите локальный сервер:

```bash
cd /Users/anso/Documents/ИИ/ege-task-generator/task19-21
python3 -m http.server 5084
```

После этого откройте:

- `http://127.0.0.1:5084`
