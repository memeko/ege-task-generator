# Генератор задания 26

Статическая страница `HTML + CSS + JS` для генерации вариантов задания 26 ЕГЭ.

## Запуск локально

```bash
cd /Users/anso/Documents/ИИ/ege-task-generator/task26
python3 -m http.server 5092
```

Открыть: `http://127.0.0.1:5092`

## Что генерируется

- несколько типов задач 26-й линии;
- входной файл в формате `.txt` для каждого варианта;
- предпросмотр первых строк данных;
- пошаговый разбор и Python-решение в спойлере.

## Логические ограничения генерации

- для гостиничных типов: этажи `<= 300`, комнаты на этаже `<= 100`;
- для остальных типов соблюдаются диапазоны параметров из условий.
